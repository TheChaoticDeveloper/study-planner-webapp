# Study Planner Web App

## Overview
The Study Planner is a web-based application designed to help students or non-students manage their academic workload by organizing tasks, tracking deadlines, and prioritizing assignments efficiently.

This project was created as a back-up for my IB personal project in-case if the 'AI study planner' didn't work.

---

## Goal
The goal of this project was to design and develop a functional and user-friendly study planner that allows students to:
- Organize academic tasks
- Track upcoming deadlines
- Prioritize work effectively
- Maintain productivity using a simple digital tool

## Features
- Add, view, and delete study tasks
- Assign subjects and priorities to tasks
- Visual priority indicators (low, medium, high)
- Tasks are saved using browser local storage
- Clean and responsive user interface
